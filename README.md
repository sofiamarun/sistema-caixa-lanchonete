# Sistema de Caixa Virtual para Lanchonete

Este projeto consiste em um sistema de PDV (Ponto de Venda) simulado, desenvolvido em Python. O objetivo foi aplicar conceitos fundamentais de lógica de programação para resolver um problema real de estabelecimentos comerciais: o gerenciamento de pedidos e fechamento de conta.

# Funcionalidades
- **Catálogo de Produtos:** Estruturação de dados para armazenar itens e preços.
- **Processamento de Pedidos:** Somatória automática dos itens selecionados pelo usuário.
- **Interação via Console:** Interface de texto intuitiva para operação do sistema.

## Tecnologias Utilizadas
- **Python 3**
- Estruturas de Dados (Listas/Dicionários)
- Estruturas Condicionais e de Repetição (`if/else`, `while/for`)
